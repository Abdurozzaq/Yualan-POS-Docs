Instalasi Yualan POS

Yualan POS adalah aplikasi kasir berbasis Laravel 12, Inertia.js, Vue.js, dan PostgreSQL.

âœ¨ Prasyarat

Pastikan software berikut sudah terinstal:

  • PHP 8.2+
  • Composer
  • Node.js 18+
  • PostgreSQL
  • Git

ðŸš€ Langkah Instalasi

# 1. Clone repository
git clone https://github.com/Abdurozzaq/Yualan.git
cd Yualan

# 2. Install dependensi Laravel
composer install

# 3. Salin file .env
cp .env.example .env

# 4. Edit konfigurasi .env
# Atur DB_DATABASE, DB_USERNAME, DB_PASSWORD sesuai PostgreSQL kamu

# 5. Generate app key
php artisan key:generate

# 6. Jalankan migrasi dan seeder
php artisan migrate --seed

# 7. Install frontend
npm install

# 8. Build frontend
npm run dev  # atau npm run build untuk produksi

# 9. Jalankan server
php artisan serve```
