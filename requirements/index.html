Yualan POS (Laravel 12) System Requirements

1. Server Requirements (PHP & Extensions)

Yualan POS menggunakan Laravel 12 yang membutuhkan:

  • PHP versi 8.2 atau lebih tinggi
  • PHP Extensions yang dibutuhkan:
    • BCMath
    • Ctype
    • Fileinfo
    • JSON
    • Mbstring
    • OpenSSL
    • PDO
    • Tokenizer
    • XML
    • Curl

2. Database Support

Laravel 12 mendukung berbagai jenis database. Untuk Yualan POS, gunakan:

  • PostgreSQL 17.3

3. Web Server

Laravel dapat dijalankan menggunakan web server berikut:

  • Apache (dengan mod_rewrite)
  • Nginx
  • Laravel Octane (untuk performa tinggi, menggunakan Swoole atau RoadRunner)

4. Composer

  • Composer versi 2.x atau lebih tinggi
  • Laravel menggunakan Composer untuk mengelola dependensi, pastikan tersedia melalui CLI.

5. Folder Permissions

Pastikan folder berikut memiliki izin tulis (writable):

  • storage/
  • bootstrap/cache/

6. Node.js & Frontend

Jika Anda menggunakan frontend modern dengan Vite, dibutuhkan:

  • Node.js versi 18.x atau lebih tinggi
  • npm atau Yarn
