iPaymu Setup untuk Yualan POS

1. Daftar Akun

  • Kunjungi: https://my.ipaymu.com atau https://sandbox.ipaymu.com untuk testing

 

2. Dapatkan API Key

  • Kunjungi: https://my.ipaymu.com atau https://sandbox.ipaymu.com untuk testing
  • Catat:
    • VA ID / Merchant Code
    • API Key
    • Endpoint:
      • Sandbox: `https://sandbox.ipaymu.com/api/`
      • Production: `https://my.ipaymu.com/api/`

 

3. Pasang API Key ke Yualan POS

  • Menu: `Settings → Tenant Info`
  • Input:
    • iPaymu API Key (VA) di input dengan `VA ID / Merchant Code`
    • iPaymu Secret Key di input dengan `API Key`
    • Setting endpoint di file `.env`: 
          - Sandbox: `https://sandbox.ipaymu.com/api/`
          - Production: `https://my.ipaymu.com/api/`
